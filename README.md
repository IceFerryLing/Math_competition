## 如何更新统计

在根目录运行（Windows PowerShell）：

```
Get-ChildItem -Recurse -File | Measure-Object
Get-ChildItem -Directory | ForEach-Object { "$($_.Name): " + (Get-ChildItem -Path $_.FullName -Recurse -File | Measure-Object).Count }
```
